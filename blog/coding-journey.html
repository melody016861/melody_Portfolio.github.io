💻 Coding Journey

這裡記錄了我的程式學習旅程，從基礎程式設計到機器學習與雲端運算的探索。

🚀 從地政到 AI：我的學習歷程

我最初的學習背景是 地政學，主要研究都市規劃與 GIS 數據應用。但在接觸 Python 與機器學習後，我開始對 AI、數據分析與雲端運算 產生濃厚興趣，於是決定深入學習並應用於專案開發。

🛠 重要技術技能

  • 📌 程式語言： Python、SQL、JavaScript
  • 📌 機器學習框架： TensorFlow、PyTorch、Detectron2
  • 📌 影像處理： OpenCV、AdelaiDet
  • 📌 地理資訊系統 (GIS)： QGIS、Google Earth Engine
  • 📌 雲端運算： AWS、Google Cloud、Docker

📂 我的程式專案

  • 🏡 AI + GIS 地政數據應用

    使用 機器學習模型與 GIS 數據，開發智慧土地管理系統，提高都市規劃決策的準確性。

    技術應用： Python、QGIS、Detectron2、地理數據處理

  • 🖼 影像辨識與自動標註工具

    利用 深度學習與 OpenCV，開發自動標註系統，以提高 AI 訓練數據的準確度與效率。

    技術應用： Python、OpenCV、TensorFlow、AdelaiDet

  • ☁️ 雲端計算與數據分析

    在 AWS 與 Google Cloud 上部署 AI 模型，處理大規模數據分析。

    技術應用： AWS、GCP、Docker、機器學習
